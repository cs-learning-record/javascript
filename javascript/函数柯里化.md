### 一、定义

函数柯里化又叫部分求值，维基百科中对柯里化 (Currying) 的定义为：

>在数学和计算机科学中，柯里化是一种将使用多个参数的函数转换成一系列使用一个参数的函数，并且返回接受余下的参数而且返回结果的新函数的技术。

例子

```

function add(a,b){
    return a + b;
}

// 执行add函数，一次传入两个参数即可
add(1,2)    //3

// 假设有一个curry函数可以做到柯里化
var addCurry = curry(add);
addCurry(1)(2)  // 3

```
