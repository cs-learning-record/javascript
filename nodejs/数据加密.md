## 前言

`crypto`模块提供了加密功能，包括对 OpenSSL 的哈希、HMAC、加密、解密、签名、以及验证功能的一整套封装。

```
const crypto = require('crypto');

const secret = 'abcdefg';
const hash = crypto.createHmac('sha256',secret).update('I Love cupcakes').digest('hex');

console.log(hash);
```
